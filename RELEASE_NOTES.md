# 枫枫子的备忘录 — Release Notes

## v1.0.0 · 2026-05-16

首个正式版本。

---

### 核心功能

#### 智能日程提取
从文字、截图、文件中一键提取日程和待办。支持粘贴会议纪要、课程表、通知邮件等任意格式，由本地大模型（Ollama）解析，理解日期相对表达（"下周三"、"明天下午"）并自动换算为具体日期。
- 文字输入：支持中英文混排
- 图片/截图：相册选取，支持 JPG / PNG
- 文件：PDF、TXT、Markdown

#### 日程与待办管理
- 事件（日期 + 时间 + 地点 + 备注）与待办（截止日 + 优先级）分类管理
- 左滑删除，右滑置顶
- 点击任意卡片在线编辑
- 置顶条目始终排在列表顶部
- 全量云同步，多设备共享

#### AI 规划助手
- 内嵌聊天界面，描述需求即可生成学习/项目/习惯养成计划
- AI 结合现有日程自动避开冲突时段
- 生成结构化草稿（事件 + 待办清单），确认后一键导入日历
- 浮动按钮随手呼出，可拖拽定位、靠边隐藏

#### 个性化推荐
- 来源：arXiv 学术论文 + GitHub 热门仓库
- 根据用户画像（研究领域 / 项目类型 / 技术技能）综合打分
- 评分维度：相关度 × 0.4 + 新鲜度 × 0.2 + 流行度 × 0.2 + 多样性 × 0.2
- 随机交替展示多来源内容，避免单一信息流
- 点击直接跳转原文，书签收藏
- 中文兴趣标签自动翻译扩展为英文搜索词

#### arXiv 学术日报
- 每日从个人推荐池中筛选未读 arXiv 论文，由 AI 生成结构化中文摘要
- Markdown 渲染，支持标题 / 加粗 / 列表
- 历史日报归档，随时回溯
- 偏好设置：推送时间、论文数量、领域分类

#### 用户画像（设置页）
- 手动添加兴趣标签（研究领域 / 项目类型 / 技术技能）
- 关键词支持中英文混写，自动翻译扩展（"大模型" → LLM / foundation model 等）
- 画像驱动推荐与日报的个性化程度

### 技术亮点

- **全本地推理**：Ollama + qwen3-vl:30b，数据不出内网
- **跨平台**：Flutter 3 + Material 3，单一代码库覆盖四端
- **云同步**：FastAPI 后端，JWT 鉴权，SQLite 持久化
- **毛玻璃导航栏**：BackdropFilter 浮动底栏，选中动画弹出效果
- **IndexedStack 导航**：切换 tab 不重建页面，保留滚动位置

---

### 已知限制

- arXiv 推荐与日报功能依赖后端服务在线，离线时不可用
- AI 规划助手需要 Ollama 服务运行（响应时间约 5–30 秒，取决于模型和硬件）
- Windows 安装包为自签名，首次安装需要信任证书或开启旁加载
